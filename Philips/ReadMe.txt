You need to have MongoDB installed.
You need to have Node JS installed.

1> Download the project from GIT.
2> Run the node_express_setup.sh using shell command. This will install express framework.
3> run following command from project root directory:
"node DummyDataInsert.js"
	- This will insert 27000 dummy data in local mongo DB.

4> Then start the Node server by following command : 
node server.js
